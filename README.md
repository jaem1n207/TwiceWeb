# Twice Web
TWICE Web is a simple website where girl group TWICE is introduced.

### Introduce
(infinite-scroll)
* Main Page
* Member Page
* Photo Page
* Video Page

### Project Stack
* JavaScript
* jQuery
* HTML
* CSS
